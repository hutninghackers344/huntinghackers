Who Develop Bluetooth Technology?


It was developed in 1994 by Ericsson Corp,a Swedish telecommunications comapny. The purpose of making bluetooth devices was to make communication in short-range , bluetooth is an wireless device. When two bluetooth devices connect , this is called pairing



How Bluetooth Security Works?

There are 3 basic security models bluetooth protocols are as follows:-



1.Non-secure mode:-There are no security applications in this mode.Such as authentication,encryption etc are not available.

2.Service level security mode:- In this mode the security is only at the service level.

3.Link-level security mode:-This mode works at the connection level.



How to be safe?

1.Disable Discoverability:- Don't make you'r bluetooth devices into Discoverability mode , because an attacker can caught you'r bluetooth devices signals easily.

2.Use Encryption:- Use Encryption in bluetooth devices because it will help you to transmit you'r data through Encryption which no one can read.

3.Authentication:- Ensure that you'r device is using encryption like Secure Simple Pairing to verify the identity they are connect to.

4.Unused Services:- If you'r bluetooth device is using multiple services , diable the service which you'r not using.
